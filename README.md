# 📊 Painel de Contratos Anômalos — Ceará Transparente

## 📌 Descrição
Este projeto consiste em um dashboard interativo desenvolvido em Streamlit para análise de contratos públicos, com foco na identificação de possíveis anomalias.

Os dados são coletados por meio de uma API pública, processados com Airflow e armazenados em PostgreSQL. O painel permite visualizar informações estratégicas sobre fornecedores e níveis de risco.

---

## 🚀 Tecnologias Utilizadas

- Python
- Streamlit
- PostgreSQL
- Apache Airflow
- Plotly
- API Ceará Transparente

---

## 📊 Funcionalidades

- 📌 Visualização de contratos por fornecedor  
- 📌 Classificação por nível de risco (ALTO, MÉDIO, BAIXO)  
- 📌 Indicadores principais (KPIs)  
- 📌 Gráficos interativos (barra e pizza)  
- 📌 Filtros dinâmicos
